使角色跟隨鼠標

  • 本技巧將拆解角色跟隨鼠標的各種態樣，透過不同的跟隨方式，能讓遊戲添增更多的變化。
  • 這些腳本示例，也可以將跟隨的目標「鼠標」換成其它的角色，呈現出追逐的效果。
  • 在「跟隨」鼠標狀態下，角色的面朝向通常要朝著鼠標。
  • 一般使用面朝(鼠標 v)向+移動()點來呈現「跟隨」較合適。
  • 如使只使用〖滑行積木〗來跟隨時，角色的面朝方向是沒有改變的。
  1. 總是緊黏著
  2. 一直跟隨在後
  3. 滿足條件才跟隨
  4. 相距夠近才跟隨
  5. 遠距跟隨，不碰觸
  6. 變速度跟隨
總是緊黏著

這種情況就如同將角色的x、y座標和鼠標的x、y座標疊合在一起。

可使用定位到(隨機 v)位置或定位到 x:() y:()來編程。

呈現出的效果就像使用滑鼠拖曳角色一樣。

角色始終在鼠標的位置。

											當@greenflag 被點擊
											重複無限次
											定位到(鼠標 v)位置
											end

或者，可使用以下腳本：

											當@greenflag 被點擊
											重複無限次
											定位到 x:(鼠標的 x) y:(鼠標的 y)
											end

📍偏移緊黏著

此腳本允許角色始終以相同的偏移距離緊黏滑鼠。

											當@greenflag 被點擊
											重複無限次
											定位到 x:(鼠標的 x) y:((鼠標的 y)+(30))//始終偏移在鼠標上方30點位置
											end
											
											當@greenflag 被點擊
											重複無限次
											定位到 x:((鼠標的 x)-(40)) y:(鼠標的 y)//始終偏移在鼠標左方40點位置
											end
一直跟隨在後

此腳本可以使角色始終跟隨鼠標。當鼠標移動位置，角色就會以固定速度跟隨。

此方式可以看到角色跟隨移動的過程。

但有個小麻煩，就是當鼠標停止移動時，角色仍會持續移動，造成角色會有顫動的現象。

											當@greenflag 被點擊
											重複無限次
											面朝(鼠標 v)向
											移動(10)點
											end

或者，可使用以下腳本：

											當@greenflag 被點擊
											重複無限次
											面朝(鼠標 v)向//如果沒有此積木，可能角色不會面朝鼠標
											滑行(0.1)秒到 (鼠標 v)位置//更改秒數可以改變跟隨速度
											end

或者，可使用以下腳本：

											當@greenflag 被點擊
											重複無限次
											滑行(0.1)秒到 x:(鼠標的 x) y:(鼠標的 y)
											end
滿足條件才跟隨

當積木中的〖布林積木〗返回值是True時，角色就會跟隨鼠標。

此方式可以改善角色在跟隨時，因鼠標停止移動，而造成顫動的情形。

											當@greenflag 被點擊
											重複無限次
											如果< ...::grey >那麼//布林值為true
											面朝(鼠標 v)向
											移動(10)點
											end
相距夠近才跟隨

此腳本是指角色與鼠標的相距在一定的範圍內時，角色將面向鼠標，跟隨移動。

此方式仍會出現角色因鼠標停止移動，而造成顫動的現象。具體改善方式，可以再加入大於指定距離時跟隨，以形成雙條件的滿足，來避免角色顫動。

以下腳本為改善後之腳本。

											當@greenflag 被點擊
											重複無限次
											如果<<(與(鼠標 v)的間距)<(100)>且<(與(鼠標 v)的間距)>(10)>>那麼//距離夠近，但又要大於一指定距離
											面朝(鼠標 v)向
											移動(10)點
											end
遠距跟隨，不碰觸

此腳本將使角色持續跟隨鼠標移動，但永遠不會碰到到鼠標。

當距離太遠時，移動靠近；當距離太近時，移動遠離。

											當@greenflag 被點擊
											重複無限次
											面朝(鼠標 v)向
											如果<(與(鼠標 v)的間距)<(50) >那麼//太近就後退
											移動(-10)點
											end
											如果<(與(鼠標 v)的間距)>(80) >那麼//太遠就靠近
											移動(10)點
											end
											end
變速度跟隨

角色跟隨滑鼠的速度會隨著它們之間距離的增加而增加，相反的，距離愈短，速度就愈慢。

											當@greenflag 被點擊
											重複無限次
											面朝(鼠標 v)向
											移動((與(鼠標 v)的間距)/(12))點//改變12數值，可以改變跟隨速度。
											end
